How to Connect Docker with Pycharm

Step 1: Install and Run Docker

I am using Mac OS for this project so all the instructions below will be Mac environment friendly. The gist of this is similar to other OS as well with few differences in terms and processes which can be easily found in the Docker site. Begin by going to the Docker site and install the software appropriate for your operating system.

  • Double-click Docker.dmg to open the installer, then drag the Docker icon to the Applications folder.
  • Double-click Docker.app in the Applications folder to start Docker. The Docker menu in the top status bar indicates that Docker Desktop is running, and accessible from a terminal.

    • Step 2: Setup Pycharm

    Once you've successfully installed Docker, start up Pycharm and go through initial configuration while making sure you are signed into your JetBrains account.

  • Create a Github Repository and clone it into your local directory using Get from Version Control on Pycharm. Select Git and provide the CODE link from your Github repository. I used the HTTPS code to clone mine.
  • Once Pycharm and Git are connected, enable Docker by going to Preferences -> Build, Execution, Deployment -> Docker. If the window is empty, select the + sign to add Docker and select Docker for Mac. As long as it says connection successful,select 'Ok'.
  • Create a new file on your project called Dockerfile
  • Add the Dockerfile code mentioned in this website to your Docker file: Dockerize with Python Application
  • Create a new directory called 'src' and add a Python file called my_script.py and add the code for the script from the above mentioned website.
  • Navigate to 'Add Configuration' and select the + button and choose Dockerfile under the Docker field.
  • Add 'Barcode' as the Name and image tag and hit apply. Press Ok to exit.
  • Set up a Docker Interpreter by navigating to the bottom right of your Pycharm window and select 'Add Interpreter'
  • Fill in the information as below image and hit Ok
  • Now that everything is set, press the green Play button next to Barcode Configuration and see your code run with Docker in Pycharm!
