Basic Linux Commands

What is Linux

Linux is an opensource UNIX based operating systems based on the Linux Kernel. Although most of the above Operating Systems come with a decent GUI, it is important to realize and utilize the potential of the Command Line Tool. Commonly referred to as Linux Shell, the command-line tool unveils access to new features that are not available through Graphical User Interface (GUI). Open terminal and try these below commands out:

  • In Terminal, to view you current directory, type pwd
  • To switch directories, type cd followed by the name of the directory.
  • To edit settings, press ctrl + ,
  • To view the content of your current directory, type ls
  • To view more content of your current directory, type ls -l. This long listing adds more information to the default ls output.
  • To view list of all files as wells as the hidden files or directories within the directory, type ls -a
  • To create a new directory, type mkdir directoryname
  • To create directory within directory instead of typing mkdir multiple times, you can type mkdir -p dir1/dir2/dir3
  • To add a file to your directory, type touch file1
  • To use a text file editor called nano to create a new file nano todo-list.txt
  • To view the newly created file, type cat todo-list.txt
  • To view the type of file is in a directory, type file name-txt.
