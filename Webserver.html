Deploy a Web server on Docker

Steps to deploy a simple Webserver with Docker on a Mac

Step 1: Make a directory

- mkdir /test

- cd /test


Create a sample web page with name index.html

- vim /test/index.html

- insert this in the file


Installing Nginx

Type in docker run -d -P -v $HOME/test:/usr/share/nginx/html \

--name (name of folder) nginx

This will install nginx which you would need for your Mac


Check the webserver

docker port test (Get's the port number)

ipconfig getifaddr en0 (Get's the IP address)

Then run type it in your browser for example: 142.23.02.2:80
